 Music Popup Add-on Setup
 This script is provided free for any type of website project. Support is not included.

[ Open Music Popup ]

View support for this addon

Music provided by Square Peach Music


STEP #1: Create Folder
Create a new folder in your main template folder and name this folder "music-pop" all lowercase.


STEP #2: Copy Files
Copy all the files from this zip file into the new "music-pop" folder.


STEP #3: (OPTION A)
Paste the following code into the "head" of your existing webpage that you want the popup on. Put the code just before the </head> near the top of the page. Make backups of your webpages before you start.




STEP #3: (OPTION B)
Instead of adding the above to each HTML page, you can instead add the following script to one of your global .js files that's used in the </head> section of your webpages like the "javascripts.js" or "pop-closeup.js". If you use this option test your menus and other webpage scripts after you add this script.




STEP #4: (OPTION A)
Adding links to open the popup.

HTML page link: Paste the following in the main body of your HTML webpage:

<a href="javascript:popmusic('music-pop/music.html')">Play Music</a><br>

Link for a menu.js: Add the following in place of the .html part of a menu link. For example, replace "about.htm" with the following code:

javascript:popmusic(\'music-pop/music.html\')

If the above does not work, you may need to use the following code in your menu.js:

javascript:popmusic('music-pop/music.html')

...or try this code:

javascript:popmusic(\"music-pop/music.html\")

Link for the sidebar: Add the following to your sidebar.js. You may need to create a new class for the link colors.

document.write('<a href="javascript:popmusic(\'music-pop/music.html\')">Play Music</a><br>');



STEP #4: (OPTION B)
For an automatic popup. Select, copy, and paste the following code to the body tag in any HTML page that you want the music popup to open with automatically.

onload="popmusic('music-pop/music.html')"

EXAMPLE: It will go in the "body" tag like the following:

<body onload="popmusic('music-pop/music.html')">


STEP #5: Replace Music
Because some browsers can not play mp3 files, you'll need to replace both the mp3 and ogg files included in this download in the "music-pop" folder.

You can optionally edit any one of the 3 "music_frame-?.html" HTML pages that you placed in the "music-pop" folder with different music file names. Edit the "music-?.mp3" code in 3 places and the "music-?.ogg" in 1 place in each "music_frame-?.html" page.


TROUBLESHOOTING:
  1. If you added the step #3B script to the "javascripts.js", make sure each page has this code near the top <script language="JavaScript" type="text/javascript" src="javascripts.js"></script> The same applies to any .js file you added the script to. It must be near the top of each HTML page.
  2. If the popup is a missing page see missing website areas.
  3. If the music is not playing see sound not working support.
  4. If the music does not play, or plays the wrong music, after editing one of the "music_frame-?.html" pages: Make sure you edited the sound file mp3 name in 3 places and ogg in 1 place, and double check your filenames and HTML code.
  5. Download the add-on again and try again from the start.

OPTIONS:

BUTTON WIDTH AND FONT COLORS:
Edit the "music.css" in your "music-pop" folder to change text or button colors or to set the width of the music popup buttons. Click for color choices.

ADDING MORE SONGS:
The "music.html" is the main music page. Each song has it's own HTML page that the "music.html" is displaying inside of an IFrame. To add a new song to the player:
  1. Make a copy of the "music_frame-1.html". Rename this to "music_frame-4.html".
  2. Edit the new "music_frame-4.html" and change the "music-1.mp3" code in 3 places and "music-1.ogg" in one place to "music-4".
  3. Place this new music .mid, .mp3, .au or .wav file in the template "music" folder.
  4. Edit the "music_frame-4.html" and change the song title.
  5. Edit the "music.html" and find the "START PLAY MENU". Copy the 2 lines that make a link and paste them under one of these links. Edit the HTML page name with your new "music_frame-4.html" link.

BUTTON NAMES:
Edit the "music.html" to change the popup button names.

SONG TITLES:
Edit each "music_frame-?.html" to change the text song title names. It may be hard to view the text in these pages because they use a transparent background. Edit the source code.

THE MUSIC PLAYERS:
The actual player code will be found in the 3 "music_frame-?.html" pages. You can edit some of the options such as the height and width of the player, border colors and other items in these 3 pages. Some of the styles are also in the "music.css".

VIDEO:
You can use these players for video. This will involve editing the code in the "music_frame-?.html" pages. We suggest searching at Google for "embed object video" for more info on the options available, or you can use YouTube embed code.

POPUP SIZE:
The size of the popup can be edited in the "pop-up-music.js" or in the code you added in step #3B above. Currently this is set to 400 x 220 pixels.
